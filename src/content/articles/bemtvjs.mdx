A BemtvJS é o meu projeto mais importante e recente. Foi um trabalho de longos
meses e, sem dúvida, um projeto de altíssima dificuldade. Durante a construção
da biblioteca, tive que tomar decisões de design para tornar a API do Bemtv
fácil de usar.

Durante o processo de desenvolvimento, precisei criar: um algoritmo de
reconciliação, um roteador SPA, uma linguagem de marcação própria (semelhante
ao HTML), uma forma de usar CSS-IN-JS de forma integrada, lidar com eventos do
DOM, e criar a documentação da API em português e inglês. Posso continuar
listando várias outras coisas porque, sem dúvida, foi meu projeto mais
desafiador e estou muito feliz com o resultado atual.
